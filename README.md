# ◆ OPC 创业者联盟 — One Person Company Alliance

> 一个人就是一家公司。在这里分享想法、寻找资源、连接同行。

🌐 **Live Site:** [https://mokangmedical.github.io/opc-alliance/](https://mokangmedical.github.io/opc-alliance/)

## 功能

### 💡 想法广场
- 发布商业想法、行业洞察、提问、经验分享
- 按类别筛选（想法/洞察/提问/经验/工具）
- 点赞、评论、收藏系统
- 标签分类

### 📚 资源库
- 分享和发现创业工具、模板、课程、书籍
- 按类别筛选（工具/模板/课程/书籍/社群/API）
- 社区投票，优质资源置顶

### 👥 联盟成员
- 成员目录，展示角色和贡献
- 搜索功能
- 积分排行榜

### 💬 讨论区
- 发起深度讨论话题
- 回复和互动
- 浏览量统计

### 🛠️ 项目展示
- 展示你的OPC项目
- 技术栈标签
- 收入展示（可选）

### 🏠 首页 Dashboard
- 实时统计（成员/想法/资源/项目）
- 最新想法流
- 每日一问
- 活跃成员排行
- 快速资源入口

## 技术栈

- 纯 HTML + CSS + JavaScript（零依赖）
- LocalStorage 本地数据持久化
- 响应式设计，支持移动端
- 暗色主题
- 中文优先界面

## 本地开发

```bash
# 克隆
git clone https://github.com/MoKangMedical/opc-alliance.git
cd opc-alliance

# 打开
open index.html
# 或
python3 -m http.server 8080
```
